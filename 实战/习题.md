以下哪一条语句不能实现"hello world"字符串在一行中输出？

```python
print("hello world")    # A
print('hello \
world')   # B
print('''hello 
world''')   # C
print('hello world')   # D
```

答案AB,     B、“\”是续行符 C、三引号可以原样输出字符串
